Detailed Description

A page (or tab) to be displayed in the options UI.

It contains a id that is used to detect what options should go into this page (see cOpt::getPage). It also has a name that will be translated and displayed. And it has items, each item is either nothing (will be represented as empty line in the UI) or the name of an option.
