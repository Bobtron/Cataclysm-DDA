134 /*
135  * List of known flags, used in both terrain.json and furniture.json.
136  * TRANSPARENT - Players and monsters can see through/past it. Also sets ter_t.transparent
137  * FLAT - Player can build and move furniture on
138  * CONTAINER - Items on this square are hidden until looted by the player
139  * PLACE_ITEM - Valid terrain for place_item() to put items on
140  * DOOR - Can be opened (used for NPC pathfinding)
141  * FLAMMABLE - Can be lit on fire
142  * FLAMMABLE_HARD - Harder to light on fire, but still possible
143  * DIGGABLE - Digging monsters, seeding monsters, digging with shovel, etc
144  * LIQUID - Blocks movement, but isn't a wall (lava, water, etc)
145  * SWIMMABLE - Player and monsters can swim through it
146  * SHARP - May do minor damage to players/monsters passing through it
147  * ROUGH - May hurt the player's feet
148  * SEALED - Can't use 'e' to retrieve items, must smash open first
149  * NOITEM - Items 'fall off' this space
150  * NO_SIGHT - When on this tile sight is reduced to 1
151  * NO_SCENT - Scent on this tile (and thus scent diffusing through it) is reduced to 0. This acts like a wall for scent
152  * MOUNTABLE - Player can fire mounted weapons from here (e.g. M2 Browning)
153  * DESTROY_ITEM - Items that land here are destroyed
154  * GOES_DOWN - Can use '>' to go down a level
155  * GOES_UP - Can use '<' to go up a level
156  * CONSOLE - Used as a computer
157  * ALARMED - Sets off an alarm if smashed
158  * SUPPORTS_ROOF - Used as a boundary for roof construction
159  * MINEABLE - Able to broken with the jackhammer/pickaxe, but does not necessarily support a roof
160  * INDOORS - Has roof over it; blocks rain, sunlight, etc.
161  * COLLAPSES - Has a roof that can collapse
162  * FLAMMABLE_ASH - Burns to ash rather than rubble.
163  * REDUCE_SCENT - Reduces scent even more, only works if also bashable
164  * FIRE_CONTAINER - Stops fire from spreading (brazier, wood stove, etc)
165  * SUPPRESS_SMOKE - Prevents smoke from fires, used by ventilated wood stoves etc
166  * PLANT - A "furniture" that grows and fruits
167  * LIQUIDCONT - Furniture that contains liquid, allows for contents to be accessed in some checks even if SEALED
168  * OPENCLOSE_INSIDE - If it's a door (with an 'open' or 'close' field), it can only be opened or closed if you're inside.
169  * PERMEABLE - Allows gases to flow through unimpeded.
170  * RAMP - Higher z-levels can be accessed from this tile
171  * EASY_DECONSTRUCT - Player can deconstruct this without tools
172  * HIDE_PLACE - Creature on this tile can't be seen by other creature not standing on adjacent tiles
173  * BLOCK_WIND - This tile will partially block wind
174  * FLAT_SURF - Furniture or terrain or vehicle part with flat hard surface (ex. table, but not chair; tree stump, etc.).
175  * ROAD - Mainly affects the speed of rollerblades
176  *
177  * Currently only used for Fungal conversions
178  * WALL - This terrain is an upright obstacle
179  * THIN_OBSTACLE - This terrain is a thin obstacle, i.e. fence
180  * ORGANIC - This furniture is partly organic
181  * FLOWER - This furniture is a flower
182  * SHRUB - This terrain is a shrub
183  * TREE - This terrain is a tree
184  * HARVESTED - This terrain has been harvested so it won't bear any fruit
185  * YOUNG - This terrain is a young tree
186  * FUNGUS - Fungal covered
187  *
188  * Furniture only:
189  * BLOCKSDOOR - This will boost map terrain's resistance to bashing if str_*_blocked is set (see map_bash_info)
190  * WORKBENCH1/WORKBENCH2/WORKBENCH3 - This is an adequate/good/great workbench for crafting. Must be paired with a workbench iexamine.
191  */
