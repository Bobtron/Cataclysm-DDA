Detailed Description

A wrapper over a pointer to a curses window.

All curses functions here receive/return and operate on such a window. The objects have shared ownership of the contained pointer (behaves like a shared pointer).

Use newwin to get a new instance. Use the default constructor to get an invalid window pointer (test for this via operator bool()). Use the operator= to reset the pointer.
