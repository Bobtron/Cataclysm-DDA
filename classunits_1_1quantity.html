inlineconstexpr

Addition and subtraction of quantities of the same unit type.

Result is a quantity with the same unit as the input. Functions are templated to allow combining quantities with different value_type, e.g.

quantity<int, foo> a = ...;
quantity<double, foo> b = ...;
auto sum = a + b;
static_assert(std::is_same<decltype(sum), quantity<double, foo>>::value);
