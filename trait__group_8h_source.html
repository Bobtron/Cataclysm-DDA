39 /**
40  * Get a trait group ID and optionally load an inlined trait group.
41  *
42  * If the value is string, it's assumed to be a trait group id and it's
43  * returned directly.
44  *
45  * If the value is a JSON object, it is loaded as a trait group. The group will be given a
46  * unique id (if the JSON object contains an id, it is ignored) and that id will be returned.
47  * If the JSON object does not contain a subtype, the given default is used.
48  *
49  * If the value is a JSON array, it is loaded as a trait group: the default_subtype will be

51  * a trait group definition (see format of trait groups).
52  *
53  * @param default_subtype If an inlined trait group is loaded this is used as the default
