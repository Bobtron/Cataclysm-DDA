Detailed Description

A point in the game time.

Use calendar::turn to get the current point. Modify it by adding/subtracting time_duration. This can be compared with the usual comparison operators. It can be (de)serialized via JSON.

Note that is does not handle variable sized season length. Changing the season length has no effect on it.
