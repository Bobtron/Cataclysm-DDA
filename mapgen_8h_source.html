205  *
206  * Instructions for adding more mapgen things:
207  * 1. create a new class inheriting from @ref jmapgen_piece.
208  * - It should have a constructor that a accepts a @ref JsonObject and initializes all it local
209  * members from it. On errors / invalid data, call @ref JsonObject::throw_error or similar.
210  * - Implement the @ref apply function to do the actual change on the map.
211  * 2. Go into @ref mapgen_function_json::setup and look for the lines with load_objects, add a new

213  * @code load_objects<your_own_class_from_step_1>( jo, "place_something" ); @endcode
214  * Use a descriptive name for "something", preferably matching the name of your class.
