Detailed Description

template<class T>
struct cata::heap< T >

This class is essentially a copyable unique pointer, like value_ptr, except it hides the fact it is a unique_ptr.

It is intended for helping make types noexcept movable by moving non-noexcept-movable types to the heap.

Contractually speaking, should never be empty. However for performance reasons in eg. item, moved-from heap<> objects do not automatically reset with a new copy of the wrapped data.
