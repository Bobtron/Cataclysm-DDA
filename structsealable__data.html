Detailed Description

There are a few implicit things about this struct when applied to pocket_data:

  • When a pocket_data::open_container == true, if it's sealed this is false.
  • When a pocket_data::watertight == false, if it's sealed this is true. This is relevant for crafting and spawned items. Example: Plastic bag with liquid in it, you need to poke a hole into it to get the liquid, and it's no longer watertight
