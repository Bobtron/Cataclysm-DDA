◆ load_tilejson_from_file()

void tileset_cache::loader::load_tilejson_from_file ( const JsonObject &  config)
private

Load tiles from json data.This expects a "tiles" array in config.

That array should contain all the tile definition that should be taken from an tileset image. Because the function only loads tile definitions for a single tileset image, only tile indices (tile_type::fg tile_type::bg) in the interval [0,size]. The offset is automatically added to the tile index. sprite offset dictates where each sprite should render in its tile

Exceptions
std::exceptionOn any error.
