Detailed Description

template<typename T, size_t kInlineCount = 4, typename SizeT = uint8_t>
struct small_literal_vector< T, kInlineCount, SizeT >

A vector type that is specialized for storing trivial types in contiguous storage with a statically predetermined amount of inline storage.

Because this is intended to be small, the internal capacity and size variables default to uint8_t for minimal overhead over the inline storage.
