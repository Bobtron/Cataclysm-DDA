bool item_pocket::favorite_settings::accepts_item ( const item &  it) const

Whether an item passes the current whitelist and blacklist filters.

Check to see if the given item is accepted by this pocket based on player configured rules.

The rules are defined in two list:

  • whitelist - when one or more items are on the list then only those items are accepted.
  • blacklist - those items listed here are never accepted.

When a whitelist is empty and the item is not blacklisted then it will be accepted. Container items will be accepted only when all items inside are accepted.

Note that the rules take into account both item id and category which have to be accepted by the rules outlined above for an item to be accepted.

Parameters
ititem to check.
Returns
true if the item is accepted by this pocket, false otherwise.
