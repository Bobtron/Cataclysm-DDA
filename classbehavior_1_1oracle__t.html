Detailed Description

An oracle is a class in charge of introspecting into a game entity in a particular way.

The base class has trivial functionality, but provides a mechanism for aggregating oracle instances regardless of type. Specialized oracles, such as a character_oracle_t, provide some number of predicates to retrieve information about the referenced entity.
