Detailed Description

template<typename T>
class pimpl< T >

This is a wrapper for implementing the pointer-to-implementation technique, see for example http://en.cppreference.com/w/cpp/language/pimpl.

It's a container for one element of type T, which is always present (one can not create a pimpl instance without a contained T instance).

The contained T can be access like a pointer via operator-> and operator.

The constructor forwards all its arguments to the constructor of T.
