174 /**
175  * A duration defined as a number of specific time units.
176  * Currently it stores the number (as integer) of turns.
177  * Note: currently variable season length is ignored by this class (N turns are
178  * always N turns) and there is no way to create an instance from units larger
179  * than days (e.g. seasons or years) and no way to convert a duration into those
180  * units directly. (You can still use @ref calendar to do this.)
181  *
182  * Operators for time points and time duration are defined as one may expect them:
183  * -duration ==> duration (inverse)
184  * point - point ==> duration (duration between to points in time)
185  * point + duration ==> point (the revers of above)
186  * point - duration ==> point (same as: point + -duration)
187  * duration + duration ==> duration
188  * duration - duration ==> duration (same as: duration + -duration)
189  * duration * scalar ==> duration (simple scaling yields a double with precise value)
190  * scalar * duration ==> duration (same as above)
191  * duration / duration ==> scalar (revers of above)
192  * duration / scalar ==> duration (same as: duration * 1/scalar)
193  * duration % duration ==> duration ("remainder" of duration / some integer)
194  * Also shortcuts: += and -= and *= and /=
195  */
