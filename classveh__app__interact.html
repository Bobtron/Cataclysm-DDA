Detailed Description

Appliance interaction UI.

Works similarly to veh_interact, but has a much simpler design. The intended usage is:

player_activity act = veh_app_interact::run( veh ); if( act ) { player_character.moves = 0; player_character.assign_activity( act ); }

The UI contains 2 parts: an upper portion showing usage info about the appliance, and a lower portion listing actions that can be performed on the appliance.

void veh_app_interact::siphon ( )
private

Function associated with the "SIPHON" action.

Checks all appliance parts for a liquid containing part that can be siphoned. If multiple parts are eligible, the player is prompted to select one. A liquid handling activity is assigned to the player to process the transfer.
