Detailed Description

template<typename Submap>
class maptile_impl< Submap >

A wrapper for a submap point.

Allows getting multiple map features (terrain, furniture etc.) without directly accessing submaps or doing multiple bounds checks and submap gets.

Templated so that we can have const and non-const version; aliases in maptile_fwd.h
