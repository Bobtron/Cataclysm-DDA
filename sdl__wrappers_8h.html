Wrappers for SDL functions that does error reporting and that accept our wrapped pointers.

Errors are reported via the usual debug log stream (exceptions are noted below).

CreateTextureFromSurface returns an empty SDL_Texture_Ptr if the function fails (the failure is also logged by the function). load_image throws if the loading fails. Its input must be a valid C-String.
