Detailed Description

template<typename T>
class list_circularizer< T >

Class used to access a list as if it were circular.

Some times you just want to have a list loop around on itself. This wrapper class allows you to do that. It requires the list to exist separately, but that also means any changes to the list get propagated (both ways).
