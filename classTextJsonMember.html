Detailed Description

Represents a member of a TextJsonObject.

This is returned when one iterates over a TextJsonObject. It is TextJsonValue, which is the value of the member, which allows one to write: for( const TextJsonMember &member : some_json_object ) TextJsonArray array = member.get_array(); }
