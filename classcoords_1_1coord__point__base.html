Detailed Description

template<typename Point>
class coords::coord_point_base< Point >

A generic coordinate-type-safe point.

Generally you wouldn't use this class directly, but via the typedefs below (point_abs_ms, etc.).

Point should be the underlying representation type (either point or tripoint).

Origin and Scale define the coordinate system for the point.

InBounds define if the point is guaranteed to be inbounds.

For more details see doc/POINTS_COORDINATES.md.
